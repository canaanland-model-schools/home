🔗 STEP 9: Connect Firebase to Your HTML

Add this inside your HTML (before your script):

<script type="module">
import { initializeApp } from "https://www.gstatic.com/firebasejs/10.12.0/firebase-app.js";
import { getFirestore, collection, addDoc } from "https://www.gstatic.com/firebasejs/10.12.0/firebase-firestore.js";

const firebaseConfig = {
  apiKey: "PASTE_HERE",
  authDomain: "PASTE_HERE",
  projectId: "PASTE_HERE"
};

const app = initializeApp(firebaseConfig);
const db = getFirestore(app);

// Save function
async function saveRecord(record) {
    await addDoc(collection(db, "examRecords"), record);
}
</script>


🔄 STEP 10: Replace localStorage in Your Code

Find this:

localStorage.setItem(...)

❌ REMOVE IT

Replace with:

await saveRecord(record);

🔐 STEP 12: Fix Firestore Rules (VERY IMPORTANT)

Go to:
👉 Firestore Database → Rules

Replace everything with:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /examRecords/{doc} {
      allow read, write: if true;
    }
  }
}

Click Publish
